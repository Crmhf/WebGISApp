Assignment 2

Leaflet map on a Bootstrap page

View project on GitHub

Requirements

Using the map from Assignment 1, create a webpage based on Bootstrap with mobile-friendly navigation. Put the CSS and JavaScript into separate files, and let the HTML file contain pure HTML. The navigation should include links to other sites.

Deliverables

A single zip file containing the files and folders, submitted through Moodle, or host the files online and submit the URL through Moodle.

Deadline

Tuesday, February 16, at 3pm.

Tips

You’ll need to create folders and files to hold the CSS and JavaScript: 

assignment2/   
  index.html    
  css/     
    starter-template.css
    style.css   
  js/
    script.js
  • Copy the JavaScript and CSS from your assignment 1 document and paste into script.js and style.css.
  • Edit the CSS so the map doesn’t take up the whole page (delete “html” and “body” selectors, set width and height to px).
  • Edit the CSS to center the map div (margin: 0 auto;).
  • The HTML page needs a title.
  • Link the Leaflet CSS and your style.css
  • Put the map div into the starter-template div.
  • Link the Leaflet JavaScript and your script.js.
  • In the navbar, change “Project name” to “Assignment 2” and link it to index.html. Edit and link “Home”, “About”, and “Contact” to something else.
